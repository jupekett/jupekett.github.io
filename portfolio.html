JavaScript & React

Othello

A game of Othello created with React. It was developed as a bonus exercise for the course TIEA2120 Web GUI programming at JYU.

Size of the gameboard can be chosen. Moves are made by drag 'n dropping the player's draggable disk into a legal square, which are highlighted on the board.

  • Deployed site
  • Code repository

Rogaining data front end

A React front end for displaying and manipulating data from a rogaining contest. This application was built as an exercise for the course TIEA2120 Web GUI programming at JYU.

Teams can be added and modified, with some input verification in place. Checkpoint codes and coordinates can be modified. Utilizes Leaflet-library with OpenStreetMap tiles to display a tiny map when clicking on checkpoint coordinates. The checkpoint marker can then be drag 'n dropped, which updates teams' points and distances.

IP-enthusiast

A pseudo-humorous web application, which leverages two existing SOAP services to convert an IP address to uppercase words. It was developed for the course TIES4560 SOA and Cloud Computing at JYU. The idea was influenced by the style of Dan Higginbotham, author of Clojure for the brave and true.

Most of the source files in com.jupekett.ties4560.task1.client were generated by JAX-WS RI, and much of the code is based on course examples.
